About this project

This project is a personal project built for Manipal University's Web Technology Lab Project Assignment Jaipur by Utkarsh Bahuguna.

It is a website for sports news updates and predictions for football, basketball, etc.

Various technologies used in building it include HTML,CSS Javascript(DOM Manipulation to insert news cards efficiently, FETCH Requests to free API's for getting prediction numbers, other minor functions making development seamless.

Frontend was built using HTML,CSS,Bootstrap and light amounts of Javascript. The backend was build using some Javascript, JSON handling through JS as well and fetch API handling through simple JS GET queries.
